# Lubriskin Landing

Proyecto listo para subir a GitHub y desplegar en Vercel.

## Pasos

1. Sube esta carpeta a un repositorio de GitHub.
2. En Vercel, crea un proyecto nuevo e importa el repo.
3. Vercel detectará Vite automáticamente.
4. En Settings > Domains, agrega el dominio que quieras usar, por ejemplo `iqlab.com.mx`.

## Comandos locales

```bash
npm install
npm run dev
npm run build
```
